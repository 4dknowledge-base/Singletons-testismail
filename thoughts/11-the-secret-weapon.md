# 11. The secret weapon

There&#8217;s one last point we can work on.
In compiled mode, code is generated with [execution control](https://developer.4d.com/docs/Concepts/interpreted-compiled). This control is generated by default in the latest versions of 4D and must be activated in older databases.
This control is necessary for code security, but it theoretically slows down execution and, in certain specific cases, it may be a good idea to deactivate it when you&#8217;re sure of your code. 

So here&#8217;s the code modified once again (in the ***clean_folderName5*** method):

```
#DECLARE($source : Text)->$result : Text var $i; $c : Integer //%R- $result:=$source $i:=1 While ($i246) $c:=Character code($result[[$i]]) Case of : ($c=34) $result[[$i]]:="'" : ($c0) $result[[$i]]:="_" End case $i:=$i+1 End while //%R+
```
